# 🔥 Daily Calorie Tracker

A simple, clean calorie and macronutrient tracking app built with Streamlit.

## Features
- 📖 180+ food items with full macro data (Protein, Carbs, Fat, Fiber)
- 📅 Date-based tracking
- 🍽️ Meal categories: Breakfast, Lunch, Dinner, Snack
- ✏️ Manual calorie & macro entry
- 📧 Email daily summary
- 📏 Unit hints (grams vs pieces) per food item

## Run Locally
```bash
pip install streamlit
streamlit run app.py
```
